Dropdown 下拉菜单 ​

下拉菜单（Dropdown）是一种常见的交互组件，在页面中提供一个下拉菜单，用户可以从中选择某项操作或选项。

基本使用 ​

通过 NueDropdown 组件创建一个基本的按钮式的下拉菜单。

通过 text 属性设置下拉菜单触发按钮中的文字。

  • Dropdown item 1
  • Dropdown item 2

<template>
    <nue-div style="height: 160px">
        <nue-dropdown text="Click to open dropdown">
            <template #dropdown>
                <li>Dropdown item 1</li>
                <li>Dropdown item 2</li>
            </template>
        </nue-dropdown>
    </nue-div>
</template>

设置触发方式 ​

通过 trigger 属性设置下拉菜单的触发方式，可选值为 hover 和 click。

  • Dropdown item 1
  • Dropdown item 2

<template>
    <nue-div style="height: 160px">
        <nue-dropdown text="Hover to open dropdown" trigger="hover">
            <template #dropdown>
                <li>Dropdown item 1</li>
                <li>Dropdown item 2</li>
            </template>
        </nue-dropdown>
    </nue-div>
</template>

设置菜单的对齐方式 ​

通过 align 属性设置下拉菜单的对齐方式，可选值为 left、center、right。

  • Dropdown item 1
  • Dropdown item 2
  • Dropdown item 1
  • Dropdown item 2
  • Dropdown item 1
  • Dropdown item 2

<template>
    <nue-div style="height: 160px" justify="space-between">
        <nue-dropdown text="Left aligned" align="left">
            <template #dropdown>
                <li>Dropdown item 1</li>
                <li>Dropdown item 2</li>
            </template>
        </nue-dropdown>
        <nue-dropdown text="Center aligned" align="center">
            <template #dropdown>
                <li>Dropdown item 1</li>
                <li>Dropdown item 2</li>
            </template>
        </nue-dropdown>
        <nue-dropdown text="Right aligned" align="right">
            <template #dropdown>
                <li>Dropdown item 1</li>
                <li>Dropdown item 2</li>
            </template>
        </nue-dropdown>
    </nue-div>
</template>

<style scoped>
.nue-dropdown li {
    width: 200px;
}
</style>

菜单选项事件处理 ​

通过事件 execute 处理下拉菜单选项的点击事件。

在 dropdown 插槽内，通过给对应的元素添加 data-executeid 属性来指定可执行选项的动作标识，并在 execute 事件所传递的 id 参数中获取该标识，根据标识执行对应的动作。

  • Coding
  • Testing
  • Maintaining

<template>
    <nue-div style="height: 160px">
        <nue-dropdown text="Click to open dropdown" @execute="handleExecute">
            <template #dropdown>
                <li data-executeid="Coding">Coding</li>
                <li data-executeid="Testing">Testing</li>
                <li data-executeid="Maintaining">Maintaining</li>
            </template>
        </nue-dropdown>
    </nue-div>
</template>

<script setup>
import { NueMessage } from "nue-ui";

const handleExecute = (id) => {
    NueMessage({
        message: `Executing option ${id}`,
    });
    // switch (id) { ... }
};
</script>

自定义下拉菜单的触发元素 ​

通过默认插槽指定下拉菜单的触发元素。

通过插槽传递出来的 clickTrigger 方法来控制下拉菜单的显示。

  • Profile
  • Logout

<template>
    <nue-div style="height: 160px">
        <nue-dropdown>
            <template #default="{ clickTrigger }">
                <nue-avatar
                    src="https://picsum.photos/200/200"
                    @click="clickTrigger" />
            </template>
            <template #dropdown>
                <li>Profile</li>
                <li>Logout</li>
            </template>
        </nue-dropdown>
    </nue-div>
</template>
