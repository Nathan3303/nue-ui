图标 ​

用于生成一个图标。

基础用法 ​

通过 NueIcon 标签生成图标。

通过 name 属性指定生成的图标。

<template>
    <nue-div>
        <nue-icon name="search"></nue-icon>
        <nue-icon name="share"></nue-icon>
        <nue-icon name="timer"></nue-icon>
        <nue-icon name="connection"></nue-icon>
    </nue-div>
</template>

大小和颜色 ​

通过 size 属性指定图标大小，接受 CSS FontSize 的合法值，默认为 16px 。

通过 color 属性指定图标颜色，接受 CSS Color 的合法值，默认为字体颜色。

<template>
    <nue-div align="center">
        <nue-icon name="search" size="24px" color="red"></nue-icon>
        <nue-icon name="share" size="24px" color="#0F0"></nue-icon>
        <nue-icon name="timer" size="24px" color="rgb(0, 0, 255)"></nue-icon>
    </nue-div>
</template>

图标自旋转 ​

通过 spin 属性指定图标是否自旋转，接受布尔类型的值，默认为 false 。

<template>
    <nue-div>
        <nue-icon name="refresh" spin></nue-icon>
        <nue-icon name="loading" spin></nue-icon>
    </nue-div>
</template>
