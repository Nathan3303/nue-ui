# NueUI

基于 Vue3 的 UI 组件库。

## 安装

```shell
npm install nue-ui
```

## 使用

::: tip
目前仅支持完全导入。
:::

```js
import { createApp } from "vue";

import NueUI from "nue-ui";
import "nue-ui/style.css";

const app = createApp(App);
app.use(NueUI);
app.mount("#app");
```
